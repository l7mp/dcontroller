# Δ-controller: Helm charts

## Setup

Register the repository with Helm.

``` console
helm repo add dcontroller https://hsnlab.github.io/dcontroller/
helm repo update
```

## Install

Deploy the Δ-controller operator.

``` console
helm install dcontroller dcontroller/dcontroller
```
